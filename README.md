This project is currently in the initial setup phase and was bootstrapped using Vite and React.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
git clone https://github.com/collinmorudi/chef-claude.git
cd chef-claude
npm install
```

### Run Development Server

```bash
npm run dev
```

Open the local development URL shown in the terminal.
